Dostępne formaty danych:


Kanał RSS umożlwia wyświetlenie danych w formatach XML oraz JSON poprzez dodanie odpowiednie rozszerzenia (np. .json) do swojego adresu.
  • xml - domyślny format danych używany dla kanału, używana gdy użytkownik nie poda formatu
    • przykład: [url]
  • json - popularny, szczególnie ostatnimi czasy, format JavaScript Object Notation.
    • przykład: [url]

Dostępne parametry:


  • limit - pozwala ograniczyć ilość wpisów do podanej wartości.
    • przykład: [url]
  • excludedAuthors (tylko kanał wpisów) - umożlwia wykluczenie poszczególnych autorów z wyników danego zapytania. Kolejni autorzy oddzieleni powinni być przecinkiem.
    • przykład: [url]
  • łączenie parametrów - oczywiście możliwe jest łączenie wspomnianych powyżej parametrów
    • przykład: [url]
